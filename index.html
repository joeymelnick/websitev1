About Me

I'm a self taught web designer and developer that is based in Boston, MA. I began to build an interest in computers at a young age of 10, attending a computer camp where I built my first website using HTML. It was basic and although I tinkered with HTML all through my childhood I never seriously persued it.

After graduating college I was fundraising for non-profits. But I was frustrated in how slow many non-profits were to adopt new technologies. I started to delve into different tech skills and learn more about email marketing. I began by re-teaching myself HTML and was instantly hooked. I quickly took up learning JavaScript and never looked back. Since then I have worked on many small projects that you can see below.

Languages and Frameworks:
Javascript, HTML5, CSS3, jQuery, Bootstrap3.
Tools & expertise:
Git, Responsive Web Design.
